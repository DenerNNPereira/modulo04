o iframe ainda pode ser usado? aula 01 capítulo 22 módulo 04

frame não é usado, só em casos muito esoecíficos. iframes são muito usados. iframe não tem uma tradução específica, mas pode ser considerado quadro em linha, quadro frame e em linha i.

para adicionar um iframe usar a tag iframe e no parâmetro src adicionar o endereço do link de um site por exemplo, neste caso o site será carregado dentro da área do iframe, parece uma aba nova, porém dentro do site. por padrão um iframe não possui borda, para alterar usar o parâmetro frameborder com valores 0 ou 1 para retirar ou adicionar. alguns sites tem uma proteção para que seus conteúdos não sejam exibidos em iframe, por questões de segurança.

configurando iframes. aula 02 capítulo 22 módulo 04

alguns dispositivos ou navegadores não são compatíveis com iframe. tudo que for digitado entre as tags iframe e barra iframe /iframe será exibido no navegador apenas se algum dispositivo não for compatível com iframes, pode colocar um texto ou um link

por padrão, a altura e largura do iframe é de 300x150 px, que pode ser alterado usando os atributos height e width dentro da tag iframe com valores entre aspas duplas que será em pixels por padrão, mas pode ser feito no css, usando um seletor e definindo o tipo de valor e valor, que será predominante sobre o inline dentro da tag iframe.

o parâmetro scolling configura barra de rolagem em um iframe, scolling="valor", valores possíveis yes sim: mesmo que não precise vai aparecer a barra de rolagem, no não: não aparece barra de rolagem mesmo que precise porém em alguns navegadores pode ou não funcionar, auto automático: se o conteúdo precisar vai ter, não precisar não vai ter a barra de rolagem

a borda pode ser configurada através do seletor em css usando as forgurações de borda border normalmente, com atributos de cor tipo expessura: border: 1px solid blue

se o dispositivo não for compatível será exibido um link ou um texto

se o dispositivo não for compatível mesmo usando configurações de tamanho inline, as configurações do seletor no css prevalecem


conteúdo local no iframe. aula 03 capítulo 22 módulo 04

para que o iframe mostre o conteúdo próprio, local, basta colocar o endereço ou link desse conteúdo ou site no atributo src. para centralizar na tela o iframe é necessário usar o display block, pois o ifrmae é inline e então usar o maring auto tudo em um seletor css. se o conteúdo do iframe tiver relação com o site que exibe o conteúdo é melhor usar o iframe dentro de uma div.
